# pestucoursework
Programming languages (Програмни езици - ПЕ) course work from third year at TU-Sofia

### Задание / Task (directly from the assistant's email) ###
Създайте приложение - <u>система за управление на хранителен блок (ученически стол)</u>. Системата има следната функционалност:

- Възможност за формиране на специфични групи потребители и начисляване на различни отстъпки и надценки за отделните групи <b>(20 точки)</b>.

- Класовете (най-малко 3 класа при реализацията) трябва да капсулира всичките детайли. Използват се private инстанции на променливите за съхраняване на различните детайли. Трябва да има най-малко два конструктора, public getters/setters за private инстанции на променливите <b>(30 точки)</b>.

Необходимо е да извършвате проверка на входните данни <b>(10 точки)</b>.

Да се предефинира операцията <<, която да се използва за извеждане на данните <b>(10 точки)</b>. Данните да се четат и съхраняват във файл <b>(20 точки)</b>.

Класовете да се опишат с UML клас диаграма <b>(10 точки)</b>.

Общ брой точки от проекта (сумирани от дадената скала) - 

Задължително данните да се въвеждат динамично, чрез меню.



Да се изготви документация, която да се предаде по време на зашитата на проекта!
